# Ejercicio Flexbox - HTML5 & CSS3

## Descripción

Proyecto realizado con **HTML5** y **CSS3** reproduciendo una interfaz usando **Flexbox** como sistema principal de maquetación.

El ejercicio incluye:
- Uso de Flexbox
- Google Fonts
- Tarjetas informativas de HTML5 y CSS3
- Diseño basado en una referencia visual
- Enlaces informativos externos

## Tecnologías utilizadas

- HTML5
- CSS3
- Flexbox
- Google Fonts
